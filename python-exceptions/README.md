## Learning Objectives

By the end of this project, you should be able to explain the following concepts without using Google:

### **General**

- Why Python programming is awesome  
- The difference between **errors** and **exceptions**  
- What exceptions are and how to use them  
- When and why we need to use exceptions  
- How to correctly handle exceptions using `try` / `except`  
- The purpose of catching exceptions  
- How to raise built-in exceptions (`raise`)  
- When and how to implement clean-up actions using `finally`  
- The importance of error handling in large systems
