Python - Test-driven development

This project focuses on mastering **Test Driven Development (TDD)** in Python.  
You will learn how to write documentation, test cases using `doctest`, and implement functions only after the tests are written.

---

## ## Learning Objectives

By the end of this project, you should be able to explain:

- Why Python programming is awesome  
- What an interactive test is  
- Why tests are important  
- How to write docstrings to create tests  
- How to write documentation for each module and function  
- What the basic option flags are to write tests using `doctest`  
- How to think about and identify edge cases  
- How to follow TDD (Test-Driven Development) principles
