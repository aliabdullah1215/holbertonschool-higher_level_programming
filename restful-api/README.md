# RESTful API Project

## Introduction
This project explores the fundamentals of RESTful APIs and how to consume and build them using different tools and technologies. It covers HTTP/HTTPS basics, API consumption using command-line tools and Python, and basic API development concepts. The goal is to understand how data is transferred between clients and servers in a structured, secure, and scalable way.

---

## Learning Objectives
By completing this project, the following concepts are covered:

- Understanding the difference between HTTP and HTTPS
- Learning the structure of HTTP requests and responses
- Consuming APIs using command-line tools such as curl
- Consuming and processing API data using Python
- Parsing JSON data and converting it into other formats (CSV)
- Understanding basic API interaction workflows

---

## Technologies Used
- Linux (Ubuntu)
- Bash
- Python 3
- curl
- requests (Python library)
- CSV module (Python standard library)
